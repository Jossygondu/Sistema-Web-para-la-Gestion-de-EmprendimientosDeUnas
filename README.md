# Sistema Web para Gestión de Servicios de Uñas

## Descripción
Este proyecto es un sistema web desarrollado para gestionar los servicios de un negocio de uñas.
Permite visualizar los servicios disponibles y agregar nuevos servicios mediante una API REST.

El sistema utiliza una arquitectura cliente-servidor donde el frontend consume información del backend.

## Tecnologías utilizadas
- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- Fetch API

## Estructura del proyecto

Sistema-Web
│
├── backend
│   └── server.js
│
└── frontend
    └── index.html

## Instalación

1. Abrir la terminal en la carpeta backend
2. Ejecutar el siguiente comando:

npm install

Esto instalará las dependencias necesarias del proyecto.

## Ejecución del sistema

Para iniciar el servidor ejecutar:

node server.js

Si el servidor inicia correctamente aparecerá:

Servidor corriendo en http://localhost:3000

## Funcionalidades
- Visualización de servicios
- Registro de nuevos servicios
- Comunicación entre frontend y backend
- Implementación de API REST
